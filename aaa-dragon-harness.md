# 《啊啊龙大冒险》开发 Harness v1

## 0. 北极星

做一个 6-12 岁小朋友和现场观众一眼能懂、10 秒能上手、60 秒能笑出来的声控跑酷游戏。

玩家不按键，靠对麦克风持续发出“啊——”给小恐龙提供能量，让它跳起、飞高并在空中漂浮。声音停，小恐龙下落。游戏必须让“喊出来”这件事成为现场表演，而不是普通输入方式的替代品。

一句话体验：

> 不喊就掉，大声会飞，喊得刚刚好才能赢。

## 1. 必须保留的核心

- 游戏名：啊啊龙大冒险
- 主角：圆滚滚、儿童节气质的小恐龙“啊啊龙”
- 输入：麦克风音量，不做语音内容识别
- 核心动作：有声音时获得持续能量、上升或漂浮，没声音时下落
- 单局时长：30-60 秒，适合现场轮流玩
- 现场展示：必须支持全屏播放
- 失败反馈：不能血腥、不能恐怖，失败是“被作业本糊脸”“掉进积木堆”“长大值爆表”这类轻松反馈
- 胜负目标：跑得越远越好，收集小红花越多越好
- 生存机制：必须有 3 条命；撞到障碍少一条命，3 次后结束；偶尔出现爱心奖励，吃到补回一条命

## 2. 目标用户

主要玩家：

- 6-12 岁小评委
- 来参加活动的 AI 社团成员
- 现场围观、愿意起哄的人

设计原则：

- 小朋友不用读说明也能玩
- 大人上台玩会显得有点笨拙、好笑、可被围观
- 观众能通过声音参与，而不是只能看屏幕

## 3. 游戏循环

标准单局流程：

1. 开始页展示啊啊龙和一句简单提示：“对着麦克风 啊——”
2. 麦克风授权
3. 2 秒安静校准
4. 2 秒大喊校准
5. 倒计时：3、2、1、啊！
6. 啊啊龙自动向右跑
7. 玩家用声音控制高度
8. 躲避障碍，收集小红花和零食
9. 三条命用完或 60 秒结束后结算
10. 展示距离、小红花、童心称号、再来一局按钮

每局必须可以在 5 秒内重开。

## 4. 输入约束

麦克风输入只使用实时音量 RMS / dB，不上传音频，不保存音频。

必须有校准：

- 安静校准：采集现场底噪
- 大喊校准：采集玩家有效喊声
- 有效阈值 = 底噪与喊声之间的动态阈值
- 阈值需要随环境噪音缓慢自适应，但不能让旁边掌声直接长期托起角色

控制映射：

- 低于阈值：啊啊龙下落
- 刚过阈值：减缓下落或小幅上升
- 中等音量：稳定飞行或悬停
- 高音量：快速上升，同时出现彩虹音浪
- 过高音量持续超过 1 秒：进入“喊太猛”状态，角色更容易飞高，但空中障碍保持少量出现

必须做输入平滑：

- 音量取短窗口平均，避免一帧尖峰误触发
- 上升力有最大值，避免一嗓子飞出屏幕
- 下落有缓冲，避免玩家换气时立即死亡
- 受伤后有短暂无敌，避免同一障碍连续扣血

必须有键盘 fallback：

- 空格键模拟声音输入
- 仅用于调试和麦克风异常时兜底
- 展示时默认隐藏键盘提示

## 5. 玩法约束

核心玩法应像 Flappy Bird + 404 小恐龙的混合体，而不是复杂平台跳跃。

障碍类型：

- 地面障碍：作业本仙人掌、闹钟怪、KPI 路牌
- 空中障碍：安静云、没收麦克风气球
- 奖励道具：小红花、棒棒糖、辣条、弹珠、游戏机

难度曲线：

- 前 10 秒：几乎不会死，用于理解控制
- 10-25 秒：出现地面障碍和奖励
- 25-45 秒：少量加入空中障碍，重点仍是让玩家享受声音悬停
- 45 秒后：速度逐步提高，但不能突然变得不公平

数值约束：

- 初始生命：3 条
- 单次撞击扣 1 条命
- 受伤后有 1 秒以上短暂无敌
- 缺命时爱心奖励低概率出现，吃到补 1 条命
- 满命时爱心极低概率出现，吃到只给额外分数或童心反馈
- 正常小朋友第一次玩，目标应能坚持 45 秒左右

禁止：

- 不做复杂关卡编辑器
- 不做长剧情
- 不做多按键操作
- 不做需要精准音高识别的玩法
- 不做需要联网才能玩的核心机制

## 6. 美术 Harness

整体风格：

- 彩色像素绘本风
- 糖果色，但不廉价
- 圆润、明亮、柔软
- 适合儿童节派对，不做暗黑、赛博、恐怖、写实

视觉关键词：

- 儿童节操场
- 彩旗
- 气球
- 积木跑道
- 小红花
- 彩虹音浪
- 零食奖励
- 作业本怪和闹钟怪

主角设计：

- 啊啊龙必须第一眼可爱
- 身体圆，四肢短，眼睛大
- 可以背小书包或戴红领巾
- 张嘴时嘴巴夸张变大
- 喊声越大，嘴巴越大，音浪越亮

画面反馈：

- 有声音：嘴巴张开，音浪出现
- 声音稳定：身体轻微漂浮，尾巴摆动
- 声音过大：脸颊鼓起，音浪过亮，出现“啊太猛了”
- 没声音：啊啊龙慌张下坠
- 撞障碍：弹性夸张动画，不做伤害感

UI 风格：

- 大按钮，少文字
- “啊啊能量”条必须清晰
- 分数和距离要大，现场后排也能看见
- 开始页、结算页都要有主角表情

颜色约束：

- 主色：薄荷绿、天空蓝、糖果红、奶油黄
- 强调色：小红花红、彩虹音浪
- 背景不得只使用单一蓝紫渐变
- 不使用大面积暗色
- 不使用过多米色/棕色，避免画面显旧

## 7. 声音与现场氛围

必须有声音反馈，但不能盖住玩家喊声。

音效：

- 起跳：短促可爱音
- 收集小红花：清亮叮声
- 撞到作业本：软弹“啪”
- 结算：派对小号或铃声

可选：

- 根据玩家声音大小改变背景音乐滤镜或音浪动画
- 全场合唱模式中，音量越大，背景彩旗晃动越明显

禁止：

- 不播放刺耳尖叫
- 不鼓励过度用嗓
- 不让玩家必须持续大喊超过 60 秒

## 8. 健康与安全

必须提示：

- “不用喊破嗓子，稳定啊就可以”
- 单局默认不超过 60 秒
- 游戏结束后鼓励换人玩

必须避免：

- 要求小朋友长时间大叫
- 用“越大声越好”作为唯一策略
- 在结算中羞辱玩家声音小

结算文案应该轻松：

- “童心发动机启动成功”
- “啊啊能量还可以更稳”
- “你把作业怪吓了一跳”

## 9. AI 使用约束

AI 用于加速创作，不作为核心在线依赖。

允许：

- 生成主视觉
- 生成角色设定
- 生成障碍物和道具素材
- 生成结算称号和搞笑文案
- 帮助写代码和调试

不依赖：

- 游戏运行时不需要调用大模型
- 不需要实时语音转文字
- 不需要在线图像生成

AI 资产必须统一风格：

- 同一角色比例
- 同一色彩体系
- 同一像素绘本质感
- 不混用写实、3D、扁平矢量和像素风

## 10. 技术约束

推荐实现：

- Web 全屏优先
- Canvas 或 PixiJS 渲染
- Web Audio API 读取麦克风音量
- 可选 Electron 打包成桌面客户端

性能目标：

- 普通笔记本稳定 60 FPS
- 麦克风输入延迟体感低于 100ms
- 首屏加载尽量低于 5 秒
- 无网络也可运行核心游戏

隐私要求：

- 麦克风数据只在本地计算音量
- 不保存、不上传、不回放音频
- 明确展示“声音只在本机用于控制”

浏览器要求：

- Chrome / Edge 优先
- 本地 file 打开如果麦克风权限受限，则使用本地服务启动

## 11. 页面结构

必须包含：

- 开始页
- 麦克风授权页或提示态
- 校准页
- 游戏页
- 结算页
- 麦克风异常兜底页

开始页文案：

- 主标题：啊啊龙大冒险
- 副标题：不喊就掉，大声会飞
- 主按钮：开始啊！

校准页文案：

- “先安静一下”
- “现在 啊—— 一声”
- “准备起飞”

结算页信息：

- 距离
- 小红花数量
- 童心称号
- 最佳啊啊能量
- 再来一局

## 12. 验收标准

功能验收：

- 不按键，只靠麦克风能完成一局
- 停止发声后角色会下落
- 声音变大后角色有明显上升反馈
- 至少有 3 种障碍和 3 种奖励
- 有完整开始、校准、游戏、结算流程
- 麦克风不可用时有明确提示和 fallback

体验验收：

- 新玩家 10 秒内理解怎么玩
- 现场后排能看懂角色状态、能量条和分数
- 一局结束后观众愿意让下一个人试
- 玩家喊声和画面反馈之间关系清楚
- 小朋友不会因为失败感到挫败

美术验收：

- 主角可爱且有记忆点
- 画面不是临时拼贴
- 障碍和奖励都能一眼识别
- 颜色明亮但不刺眼
- 截图能让人看出这是儿童节声控游戏

稳定性验收：

- 现场底噪下不会自动一直飞
- 玩家换气时不会立刻死亡
- 连续玩 10 局不会明显卡顿
- 刷新页面后能重新授权并开始

## 13. 展示策略

上台展示脚本：

1. “这个游戏不用手，只用童心发动机。”
2. “小朋友只要对着麦克风 啊——，啊啊龙就会飞。”
3. “停下来，它就会掉。”
4. “但是喊太猛也不行，会撞到安静云。”
5. 邀请小评委试玩 30 秒。
6. 最后一轮开启全场合唱模式。

展示时优先演示：

- 校准
- 一次小声飞
- 一次大声冲高
- 一次故意停声下落
- 一次收集小红花
- 一次撞作业本怪

## 14. 第一版开发范围

MVP 必做：

- 横版自动跑酷
- 麦克风音量控制跳跃/飞行
- 动态音量条
- 3 种障碍
- 3 种奖励
- 分数、距离、小红花
- 开始、校准、结算
- 可爱主角和统一背景

第一版不做：

- 多人联网
- 排行榜
- 复杂 AI 生成关卡
- 语音识别
- 长剧情
- 账号系统

可选增强：

- 全场合唱模式
- Electron 客户端
- 小评委专属结算称号
- 摄像头表情彩蛋

## 15. 图像生成 Prompt Harness

主视觉 prompt：

> 彩色像素绘本风横版跑酷游戏截图，一只圆滚滚可爱小恐龙戴着小书包，张大嘴发出彩虹音浪，在儿童节操场上奔跑，地面是彩色积木跑道，周围有小红花、棒棒糖、弹珠和游戏机奖励，道路上有可爱的作业本怪和闹钟怪，天空有气球和彩旗，明亮、童趣、精致、适合 6-12 岁儿童，游戏 UI 清晰，有声音能量条，high quality pixel storybook game art

角色设定 prompt：

> 可爱圆滚滚小恐龙游戏角色设定图，儿童节主题，戴小书包，表情包括正常、张嘴喊啊、用力喊、惊慌下落、开心收集小红花，彩色像素绘本风，干净背景，统一比例，适合横版跑酷游戏 sprite sheet

道具表 prompt：

> 儿童节横版跑酷游戏道具和障碍物 sprite sheet，包含小红花、棒棒糖、辣条、弹珠、游戏机、作业本怪、闹钟怪、KPI 路牌、安静云、没收麦克风气球，彩色像素绘本风，圆润可爱，透明背景，统一光照，适合儿童游戏

## 16. 最重要的取舍

宁可玩法简单，也要声音反馈稳定。

宁可素材少，也要画风统一。

宁可一局短，也要现场想再玩。

宁可不展示复杂 AI，也要让小朋友真的控制住啊啊龙。
